# Documentación de Ingeniería — Sistema Bowtie

Esta carpeta contiene los entregables formales del **desarrollo de
ingeniería** del sistema Bowtie. Los documentos están escritos en formato
Markdown y los diagramas en notación **Mermaid** (UML 2.5.1), que GitHub
renderiza nativamente en blanco y negro mediante el tema *neutral*.
